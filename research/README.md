## Key Concepts

- **Ordinals Protocol** — Casey Rodarmor's protocol assigning sequential integers to Bitcoin satoshis/inscriptions
- **Cursed Inscriptions** — ~472,043 inscriptions created before block 824,544 with non-standard methods; rehabilitated by Jubilee update (Jan 6, 2024)
- **secp256k1** — Bitcoin's elliptic curve signature algorithm
- **BRC-20** — Experimental fungible token standard on Bitcoin Ordinals
- **SNS (Satoshi Name Service)** — JSON-based domain name inscriptions on Bitcoin Ordinals
- **DID (Decentralized Identifier)** — W3C standard for self-sovereign identity
- **ERC-8004** — Ethereum standard for AI agent identity (January 2026)
